# 噴漆補強欄位清單

## 製程背景 process_context

| 欄位 | 代表意義 |
|---|---|
| `batch_id` | 生產批次 |
| `product_id` | 產品 ID |
| `part_id` | 零件 ID |
| `recipe_name` | 噴漆配方 |
| `paint_type` | 漆料類型，例如 primer / topcoat / gold_paint |
| `paint_batch_id` | 漆料批號 |

## 噴漆設備 paint_equipment

| 欄位 | 代表意義 |
|---|---|
| `nozzle_id` | 噴嘴 ID |
| `spray_gun_id` | 噴槍 ID |
| `nozzle_status` | 噴嘴狀態 |
| `spray_gun_status` | 噴槍狀態 |

## 噴漆設備相關 metrics

| 欄位 | 代表意義 |
|---|---|
| `paint_pressure_bar` | 漆料壓力 |
| `paint_flow_rate_ml_min` | 漆料流量 |
| `atomizing_pressure_bar` | 霧化空氣壓力 |
| `air_flow_rate_value` | 空氣流量數值 |
| `air_flow_rate_unit` | 空氣流量單位 |

## 噴塗品質與環境 quality_and_environment

| 欄位 | 代表意義 |
|---|---|
| `chamber_temperature_c` | 腔體溫度 |
| `chamber_humidity_pct` | 腔體濕度 |
| `film_thickness_um` | 膜厚 |
| `color_difference_delta_e` | 色差 |
| `defect_count` | 缺陷數 |
| `defect_type` | 缺陷種類 |
