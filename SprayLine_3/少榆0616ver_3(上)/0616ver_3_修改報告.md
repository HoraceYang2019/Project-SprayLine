## 二、本版完成事項

### 1. 版本合併

已合併 0614 上下包，不再拆包。

### 2. FutureService 對齊 Database/versionB

檔案：

```text
webservices/future_service/future_service.py
```

確認 payload 欄位對齊 `future_prediction_result`：

```text
prediction_id
batch_id
station_id
prediction_time
predicted_ok_rate
predicted_ng_count
quality_score
risk_level
model_input_source

```

```text

build_and_save_future_prediction_result()
```



### 4. Alert 寫入補 cause / response link

檔案：

```text
webservices/monitoring_worker/alert_event_writer.py
webservices/monitoring_worker/detection_mapping.py
rules/sensor_event_mapping.json
```

寫入 alert 時使用：

```text
db_alert.insert_alert_event
db_alert.link_alert_cause
db_alert.link_alert_response
```

`alert_event.cause` 暫放 `cause_catalog.cause_id`，例如：

```text
FILTER_CLOG
NOZZLE_CLOG
AIR_PRESSURE_UNSTABLE
SPRAY_WIDTH_DEVIATION
```

注意：此點仍待余宇承確認，但目前 Database/versionB/db_alert.py 的 docstring 也是這個方向。

### 5. batch_station_status 改成完整 snapshot

檔案：

```text
webservices/monitoring_worker/batch_station_status_writer.py
webservices/monitoring_worker/detection_mapping.py
```

0616ver_1 只更新單一 component state，可能覆蓋其他欄位。0616ver_3 改成：

```text
先呼叫 db_status.get_batch_station_status()
再 merge 本次 detected result
最後呼叫 db_status.upsert_batch_station_status()
```

寫入 record 會包含 6 個 state 與 6 個 response FK：

```text
robot_arm_state
nozzle_state
filter_state
compressor_state
spray_width_state
quality_state
robot_arm_response_id
nozzle_response_id
filter_response_id
compressor_response_id
spray_width_response_id
quality_response_id
```

### 6. Threshold 保留 JSON config

檔案：

```text
rules/sensor_thresholds.json
webservices/event_rule_service/event_rule_service.py
```

少榆端仍以 `rules/sensor_thresholds.json` 作為 EventRule / Monitoring 判斷來源。`Database/versionB` 內的 `sensor_threshold` table/function 先視為 DB 端保留備用。

### 7. 新增 sensor event mapping

新增檔案：

```text
rules/sensor_event_mapping.json
webservices/monitoring_worker/detection_mapping.py
```

用途是把 sensor threshold 結果補成 DB / troubleshooting 可用的結構：

```text
severity_state = warning / fault
issue_state / fault_state = 具體問題
cause_id = cause_catalog ID
response_ids = response_catalog ID list
state_field = batch_station_status 的 component state 欄位
response_field = batch_station_status 的 response FK 欄位
```
