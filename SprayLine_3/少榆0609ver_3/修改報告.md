# 0609ver_3 修改報告

## 一、本版修改目的

本版以 `0609ver_2` 為基礎，針對兩件事補強：

1. 將林聖堯 DataPreprocess 端提供的門檻數值整理進最新版。
2. 檢查並補強 ontology 的上下級關係、主要關聯與可展示路徑。

同時依新要求移除 `discussion_source`、`討論來源` 等字詞與狀態標籤，改用中性描述。

## 五、Ontology 補強

```text

```

補強 EventRule 與 threshold reference 的關係：

```text
EventRule → usesThresholdReference → SensorThresholdReference
SensorThresholdReference → appliesToSignal → SensorSignal
SensorThresholdReference → appliesToStation → Station
```

新增 ontology 檢查報告：

```text
docs/validation/0609ver_3_ontology檢查報告.md
```

## 六、Placeholder 與文字標籤調整

本版減少不必要的空白佔位文字。

處理方式：

```text
已有來源者：使用 reference value。
尚未確認者：使用 null、空值或 pending_confirmation。
狀態標籤：使用 discussion_draft、pending_confirmation、preprocess_reference_pending_confirmation 等中性描述。
```

已移除或替換特定來源稱呼。
