# 0609ver_3 Placeholder 使用原則

本版已減少不必要的空白佔位文字。

## 原則

1. 已有來源的欄位，例如 DataPreprocess 門檻值，改用 reference row 呈現。
2. 尚未取得正式資料的欄位，不填假數值，改用 `null`、空值或 `pending_confirmation`。
3. Template 只保留資料格式示意，不作為正式資料輸出。
4. 文件與欄位狀態不使用「discussion_source / 討論來源」字樣，改用中性狀態，例如 `discussion_draft`、`pending_confirmation`、`preprocess_reference_pending_confirmation`。
